Neoplastic Composition

An interactive demonstration of the CSS Box Model

Instructions

This neoplastic composition is built using the CSS Box Model. Each colored rectangle is a separate div with its own position, size, and color properties.

Box Model Components:

  • Content: The inner part of the box (colored area)
  • Padding: Space between content and border (not used here)
  • Border: Edges around the box (black lines)
  • Margin: Space between boxes (handled with position: absolute)

To position the boxes correctly:

  • position: absolute for precise placement
  • top, right, bottom, left for positioning
  • width and height for dimensions
  • border for the characteristic black lines
  • transform: translate(-50%, -50%) for perfect centering

Use the controls below to modify the composition!

Tip

Click on any box to select it, then choose a color to change it. You can also adjust all boxes simultaneously using the size and border controls.

How the Boxes Are Positioned

Each box uses CSS properties like position: absolute, top, left, width, height and border to create the neoplastic style. Positioning is relative to the parent element with class .neoplastic-composition which has position: relative.
